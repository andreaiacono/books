Settings

Configure GitHub integration to enable adding books from the browser.

GitHub Integration

The GitHub repo where your library data lives (e.g. alice/my-books)

Create at github.com/settings/tokens — scope: public_repo (or repo for private repos). Stored only in your browser's localStorage, never in the repo.

About

Biblio is a fully static personal library manager. All data lives in your GitHub repository as JSON and CSV files.

No backend. No database. No account required beyond GitHub.

Data files: data/books.json · data/books/*.json · data/reading-log.csv
